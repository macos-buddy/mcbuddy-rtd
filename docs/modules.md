## Core

The built-in core module provides actions to:

- Manage mcBuddy itself (cf. the `Santuary` and `Forbidden City`).
- Manage modules.
- Perform common tasks (e.g. install/upgrade/uninstall, configure/reset, backup/restore, clean)
